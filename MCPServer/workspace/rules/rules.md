# Правила работы агентов

1. Каждый агент общается с ядром только через stdin/stdout построчным JSON.
2. Формат входящего сообщения: `{"type": "user_input", "text": "...", "memory_snapshot": [...]}`.
3. Формат исходящего сообщения: `{"type": "agent_response", "agent": "<имя>", "text": "..."}`.
4. Агент не должен писать в stdout ничего, кроме корректного JSON (логи — в stderr или в файл).
5. Если агент получает `{"type": "shutdown"}` — он должен корректно завершиться.
6. Долгие операции (запросы к Ollama, внешним API) не должны блокировать чтение stdin дольше таймаута, заданного в конфиге оркестратора.
